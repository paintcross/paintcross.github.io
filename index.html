How to Install (.deb, .rpm, AppImage)
Linux Installation Guide
Debian / Ubuntu (.deb)

For Ubuntu, Debian, Linux Mint, and other Debian-based distributions. You can install it via graphical software manager or run the command below in your terminal:

sudo apt install ./Paint.Cross_*.deb
Fedora / Red Hat (.rpm)

For Fedora, Red Hat Enterprise Linux (RHEL), CentOS, and other RPM-based distributions. Run the command below in your terminal to install:

sudo dnf install ./Paint.Cross-*.rpm
AppImage (Universal)

A universal package that runs on almost any Linux distribution. Download the file, make it executable, and run it directly:

chmod +x Paint.Cross_*.AppImage ./Paint.Cross_*.AppImage

What can I open?

Open common image files like JPG, JPEG, and WebP, and freely export them to other formats such as PNG.

Can I save layers?

Yes. Save .paintx project files that preserve layers and edit history, then continue whenever you want.

Do I need an account?

No. There is no signup or login. Download the native app and start right away.
